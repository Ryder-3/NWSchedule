# Usage

## How to run

Ensure the latest version of react is installed

In a terminal run:

> npm install react-bootstrap
>
> npm install \\
> @fullcalendar/core \\
> @fullcalendar/daygrid \\
> @fullcalendar/timegrid \\
> @fullcalendar/interaction \\
> @fullcalendar/react
>
> npm run dev

Then, go to http://localhost:5173/ in a browser

## To end

Hit Ctrl-C in the terminal
